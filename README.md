# PPLA_HSRM – Quiz Trainer (PPL-A)

## Contents
This program is an offline quiz trainer for the PPL-A question catalogue.

---

## System Requirements
- Windows 10 or Windows 11 (64-bit)

---

## Installation (Setup)
1. Start the setup by double-clicking it.
2. Choose the installation path (default is recommended).
3. Optional: Create a desktop shortcut.
4. Complete the installation.

---

## Notes

### Windows SmartScreen Warning
If Windows shows a warning when starting the program:
1. Click **More info**
2. Then click **Run anyway**

(This warning can appear if the program is not digitally signed.)

### Program won’t start / closes immediately
- Please run as Administrator (right-click → **Run as administrator**)
- Check whether an antivirus program is blocking the EXE
- If necessary, add the folder as an exception
